# Νέα Χαρακτηριστικά Κύριας Οθόνης

## Αποτίμηση Χαρτοφυλακίου με Επιλογή Νομίσματος

- Εμφάνιση της συνολικής αξίας του χαρτοφυλακίου
- Επιλογή προβολής σε **USD ($)** ή **EUR (€)**
- Αυτόματη μετατροπή με βάση την τρέχουσα ισοτιμία EUR/USD
- Η βάση είναι το δολάριο (USD)

## Γράφημα Εξέλιξης Χαρτοφυλακίου

- Γραμμικό γράφημα που δείχνει την εξέλιξη της αξίας του χαρτοφυλακίου
- Επιλογές χρονικής περιόδου:
  - **Day** (Ημέρα)
  - **Week** (Εβδομάδα)
  - **Month** (Μήνας)
  - **Year** (Χρόνος)
- Απεικόνιση με ομαλή καμπύλη (bezier curve)

## Πίτα Κατανομής Μετοχών

- Στρογγυλό γράφημα πίτας που δείχνει την ποσοστιαία κατανομή των μετοχών
- Χρωματική κωδικοποίηση για κάθε μετοχή
- Εμφάνιση συμβόλου και ποσοστού για κάθε μετοχή
- Legend με λεπτομέρειες για κάθε μετοχή

## Γράφημα Ισοτιμίας EUR/USD

- Γραμμικό γράφημα με την ιστορική εξέλιξη της ισοτιμίας
- Εμφάνιση των τελευταίων 30 ημερών
- Επιλογή εναλλαγής μεταξύ:
  - **EUR/USD** (πόσα δολάρια για 1 ευρώ)
  - **USD/EUR** (πόσα ευρώ για 1 δολάριο)
- Εμφάνιση της τρέχουσας ισοτιμίας

## Τεχνικές Λεπτομέρειες

### API Ισοτιμιών

- Χρήση δωρεάν API: `exchangerate-api.com`
- Cache των ισοτιμιών για 1 ώρα
- Fallback τιμή σε περίπτωση αποτυχίας

### Βιβλιοθήκες Γραφημάτων

- `react-native-chart-kit`: Για όλα τα γραφήματα
- `react-native-svg`: Απαραίτητο dependency για τα γραφήματα

### Δομή Αρχείων

- `exchangeRateService.js`: Υπηρεσίες για ισοτιμίες
- `PortfolioChart.js`: Γράφημα εξέλιξης χαρτοφυλακίου
- `PortfolioPieChart.js`: Πίτα κατανομής μετοχών
- `ExchangeRateChart.js`: Γράφημα ισοτιμίας
- `mainscreen.js`: Κύρια οθόνη με ενσωμάτωση όλων των components

## Χρήση

Όλα τα νέα χαρακτηριστικά εμφανίζονται αυτόματα στην κύρια οθόνη. Το χαρτοφυλάκιο υπολογίζεται αυτόματα από τα assets που έχετε προσθέσει.
